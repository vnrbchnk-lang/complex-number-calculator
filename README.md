# Калькулятор комплексных чисел

Учебный проект по дисциплине «Инженерия тестирования».
Вариант №2 — калькулятор для комплексных чисел.

## Описание

Программное обеспечение, предоставляющее пользователю интерфейс для
выполнения операций над комплексными числами:

- Сложение
- Вычитание
- Умножение
- Деление (с проверкой деления на ноль)
- Сравнение на равенство (`==`)
- Сравнение по модулю (`<`, `>`, `≤`, `≥`)

## Технологии

- C# 12
- .NET 8.0
- Visual Studio 2022
